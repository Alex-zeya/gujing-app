# 股镜 QA 测试记录

更新日期：2026-06-12

## 测试范围

- 后端单元测试：持仓、交易流水、组合风险、行情波动雷达、新闻缓存、数据状态、搜索。
- 前端构建：React/Vite production build。
- iOS 同步：Capacitor web assets 同步到 Xcode 工程。
- 浏览器冒烟测试：登录页、Apple/微信入口提示、首页、找股票、持仓、观察池、我的、隐私政策入口。

## 必测流程

1. 登录
   - 网页预览点击 Apple 登录，应提示需要在 iPhone App 中使用。
   - 真机 App 点击 Apple 登录，应弹出系统授权面板。
   - 微信登录在开放平台配置前应给出明确提示。
   - 退出登录后回到登录页。

2. 找股票
   - 搜索代码：`600519`
   - 搜索中文：`电气`
   - 从联想列表选择股票。
   - 查看分析、K 线、加入观察池、加入持仓。

3. 持仓
   - 添加一只股票。
   - 执行买入。
   - 执行卖出。
   - 检查本金、当前市值、未实现收益、已实现收益、组合风险是否刷新。

4. 观察池
   - 查看观察股票列表。
   - 打开通知中心。
   - 打开通知设置。
   - 删除观察股票。

5. 我的
   - 修改风险偏好和昵称。
   - 打开风险免责声明。
   - 打开隐私说明和完整隐私政策网页。
   - 登录状态下验证“删除账户 / 清除数据”确认流程。

## 已知限制

- 免费行情源可能延迟或偶发字段缺失，生产版需要稳定授权数据源。
- Render 免费实例会冷启动，首次请求可能慢。
- 微信登录尚未接入正式开放平台。
- 手机号短信仅保留为后续备用能力，首版不展示短信登录入口。
